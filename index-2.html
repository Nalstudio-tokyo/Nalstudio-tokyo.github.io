Q. なぜ様々なジャンルを学ぶのか？
NAL STUDIOでは、K-POPダンスを学ぶことはアイドルを目指す上で必須だと考えています。K-POPには韓国独自の基礎があり、それを学ぶことで、どんなジャンルの振付でもグループとして揃ったダンスを踊ることができるようになります。
K-POPの要素だけでなく、多角的な視点も大切にしています。
HIPHOP・R&B・GIRLS HIPHOPなどを学ぶことで、グルーヴ・音の取り方・表現力・個性を身につけることができます。そのためNAL STUDIOでは、第一線で活躍するダンサーや振付師をインストラクターとして起用しています。ダンスを武器にしたい方におすすめです。

NAL TRAINEE
韓国式育成プログラム
NAL TRAINEEでは、ダンスを習うだけではなく、基礎力・表現力・現場対応力・パフォーマンス力・チームワークなど、実際のアイドル活動やアーティスト活動に必要なスキルを段階的に学んでいきます。
入会の流れ
クラスに参加するには2つの流れがございます。
01
スキルチェックを受ける
現在のレベルを確認し、A・B・Cいずれかのクラスへご案内します。事前に基礎動画をお渡しし、「基礎トレーニング」「得意な振付」をチェックさせていただきます。
02
スキルチェックを受けずに入会する
スキルチェックを受けない場合は、C CLASSからスタート可能です。未経験者や初心者の方も安心してご参加いただけます。
基礎トレーニング
NAL TRAINEEでは、韓国大手芸能事務所のトレーニングを参考に構築した基礎プログラムを実施しています。姿勢・リズム・体の使い方など、すべてのレベルの土台となる力を身につけます。
CLASS C ／ 基礎習得クラス
短い振付を丁寧に学びながら、1から習得していきます。未経験者や初心者の方におすすめのクラスです。
  • K-POP特有の身体の使い方
  • リズムの取り方
  • 魅せ方
→
CLASS B ／ 応用・表現クラス
1コーラス程度の振付を約3週間で習得し、より完成度の高いパフォーマンスを目指します。
  • 細かな動きの再現
  • 表現力
  • 魅せ方
  • 楽曲理解
K-POPグループのパフォーマンスに欠かせない「動きを揃える意識」も学び、周りとタイミングや動きを合わせながら踊る力を身につけていきます。
→
CLASS A ／ 実践・パフォーマンスクラス
NAL TRAINEE最高レベルのクラス。毎月の動画撮影に加え、フォーメーションを含む実践的なパフォーマンス制作を行います。
  • フォーメーション
  • チームパフォーマンス
  • カメラ映り
  • 現場対応力
  • 表情管理
案件がある際は、ダミーダンサー・バックダンサー・撮影案件などへの参加機会も予定しています。「上手に踊れる」だけでなく、「ダンスが特技です」と自信を持って言えるレベルを目指します。
